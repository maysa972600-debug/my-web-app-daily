# Smart Task Planner

This project is a web-based application designed to help users organize their daily tasks using simple AI techniques.

Users can enter a task, and the system will generate a structured plan along with a suggested duration. The system is based on a dataset that includes tasks related to study, work, health, fitness, assignments, and daily routines.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Task input
- Automatic plan generation
- Daily and structured scheduling
- Simple AI-based logic

## Purpose
The goal of this project is to demonstrate how artificial intelligence can be integrated into a web application to improve productivity and task management.
